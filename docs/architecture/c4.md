# C4 Architecture

## System Context

```mermaid
C4Context
    title System Context - Inbound Carrier Sales Automation

    Person(carrier, "Carrier", "Calls to find and book available freight loads.")
    Person(salesRep, "Sales Rep", "Receives qualified transfers after price agreement.")
    Person(brokerUser, "Broker Ops User", "Reviews call outcomes, offers, and dashboard metrics.")

    System(happyRobot, "HappyRobot Platform", "Inbound web-call agent that speaks with carriers and calls external APIs.")
    System(apiSystem, "Carrier Sales API", "API-first backend for carrier verification, load search, negotiation, call capture, and metrics.")
    System_Ext(fmcsa, "FMCSA API", "External carrier authority and eligibility data source.")

    Rel(carrier, happyRobot, "Starts inbound web call")
    Rel(happyRobot, apiSystem, "Calls authenticated API endpoints", "HTTPS + x-api-key")
    Rel(apiSystem, fmcsa, "Verifies MC number", "HTTPS")
    Rel(apiSystem, salesRep, "Returns mocked transfer success when load is agreed")
    Rel(brokerUser, apiSystem, "Views dashboard and call metrics", "HTTPS")
```

## Container Diagram

```mermaid
C4Container
    title Container Diagram - Cloudflare MVP

    Person(carrier, "Carrier", "Calls through HappyRobot web-call trigger.")
    Person(brokerUser, "Broker Ops User", "Reviews operational metrics.")
    System_Ext(happyRobot, "HappyRobot Platform", "Conversation workflow and telephony/web-call interface.")
    System_Ext(fmcsa, "FMCSA API", "Carrier authority verification.")

    System_Boundary(systemBoundary, "Carrier Sales Automation") {
        Container(worker, "Cloudflare Worker", "TypeScript + Hono", "Serves API routes, dashboard routes, auth middleware, and business orchestration.")
        ContainerDb(d1, "Cloudflare D1", "SQLite", "Stores loads, carriers, calls, offers, outcomes, and sentiment labels.")
        Container(dashboard, "Dashboard UI", "HTML/TypeScript", "Simple metrics view served by the Worker.")
    }

    Rel(happyRobot, worker, "POST/GET API calls", "HTTPS + x-api-key")
    Rel(worker, fmcsa, "Carrier verification lookup", "HTTPS")
    Rel(worker, d1, "Reads/writes relational data", "D1 binding")
    Rel(brokerUser, dashboard, "Views metrics", "HTTPS")
    Rel(dashboard, worker, "Fetches metrics API", "HTTPS")
```

## Component Diagram

```mermaid
C4Component
    title Component Diagram - Cloudflare Worker

    Container_Boundary(worker, "Cloudflare Worker") {
        Component(auth, "API Key Middleware", "Hono middleware", "Rejects unauthenticated API requests.")
        Component(loads, "Load Search Routes", "HTTP handlers", "Searches viable loads by lane, equipment, timing, and availability.")
        Component(carriers, "Carrier Verification Routes", "HTTP handlers", "Validates MC number using FMCSA adapter with mock fallback.")
        Component(negotiation, "Negotiation Engine", "Domain service", "Evaluates counteroffers for up to three rounds.")
        Component(calls, "Call Capture Routes", "HTTP handlers", "Stores extracted call data, offers, outcomes, and sentiment.")
        Component(metrics, "Metrics Routes", "HTTP handlers", "Aggregates dashboard metrics.")
        Component(repos, "Repositories", "D1 data access", "Encapsulates SQL for loads, calls, carriers, and offers.")
    }

    ContainerDb(d1, "Cloudflare D1", "SQLite", "Application persistence.")
    System_Ext(fmcsa, "FMCSA API", "External carrier eligibility source.")
    System_Ext(happyRobot, "HappyRobot Platform", "Agent workflow caller.")
    Person(brokerUser, "Broker Ops User", "Dashboard user.")

    Rel(happyRobot, auth, "Calls API")
    Rel(auth, loads, "Allows authenticated requests")
    Rel(auth, carriers, "Allows authenticated requests")
    Rel(auth, calls, "Allows authenticated requests")
    Rel(loads, repos, "Queries loads")
    Rel(carriers, fmcsa, "Verifies MC number")
    Rel(carriers, repos, "Stores verification result")
    Rel(calls, negotiation, "Evaluates offers")
    Rel(calls, repos, "Persists call outcome")
    Rel(metrics, repos, "Aggregates metrics")
    Rel(brokerUser, metrics, "Reads dashboard data")
    Rel(repos, d1, "Executes SQL")
```

## Notes

- The production deployment target is Cloudflare Workers + D1.
- Workers VPC is intentionally excluded because no private external service is required.
- Docker can be added as a local reproducibility artifact without changing the production architecture.
- API-first TDD keeps the HappyRobot integration contract stable while persistence evolves.
